 # C++ Student Management System
 
This project is a simple console-based application written in C++.
It was created to practice object oriented programming concepts and basic file handling.

Features
- Add students
- Display student records
- Store student data using files

Concepts Used
- Classes and objects
- Encapsulation
- Vectors
- Functions
- File input/output
- Input validation

What I Learned
Working on this project helped me better understand how to organize code using classes
and how data can persist using files. It also improved my problem-solving skills
and reinforced the importance of writing clear and structured code.

Future Improvements
- Add unique student IDs
- Allow editing and deleting records
- Improve error handling
